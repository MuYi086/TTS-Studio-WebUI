# Vue 工作台待完成与待验收事项

本文区分“已完成离线端到端回归”“仍需真实后端人工回归”与“设计扩展尚未实现”；静态代码存在不等于在线能力已验收。

## 已完成离线端到端回归

- 顶部工程工具已在所有主标签页可见，支持完整工程 JSON 导入导出和 `TXT`（纯文本文件）导入；实现位于 [`ProjectTransferPanel.vue`](src/components/ProjectTransferPanel.vue) 与 [`AppShell.vue`](src/app/AppShell.vue)。
- 脚本工作台已接入单行/批量生成、顺序播放、`SRT` / `WAV` / `MP4` 导出；编排位于 [`useScriptAudioRuntime.ts`](src/composables/useScriptAudioRuntime.ts)。
- 音色、SFX（场景音效）和 BGM（背景音乐）已接入真实试听、进度、波形绘制与裁剪手柄；通用运行时位于 [`useLibraryAudioPreview.ts`](src/composables/useLibraryAudioPreview.ts)，展示组件位于 [`LibraryAudioPreview.vue`](src/components/library/LibraryAudioPreview.vue)。SFX/BGM 的 `trimStart` / `trimEnd` 保存后会继续用于脚本播放与离线导出。
- 固定工程可由 [`scripts/validate-p1-regression-fixture.mjs`](../scripts/validate-p1-regression-fixture.mjs) 校验结构。
- 2026-07-16 已在 Chrome for Testing `149.0.7827.55` 中完成固定工程导入、刷新恢复、三类试听、JSON/SRT/WAV/MP4 下载回归；MP4 重新加载后确认同时包含音视频轨。环境与结果见 [`docs/本地开发与回归.md`](../docs/本地开发与回归.md)。

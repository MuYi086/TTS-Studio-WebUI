# 本地开发与回归

## 正确入口

当前 Vue 工作台位于 [`webUI/`](../webUI/)，其 `webUI/index.html` 加载 `src/app/main.ts`。根目录 [`index.html`](../index.html) 仅是旧版单文件实现的行为对照页。固定工程样例是 [`fixtures/p1-minimal-regression-project-v3.json`](../fixtures/p1-minimal-regression-project-v3.json)。

## 开发命令

```bash
cd webUI
npm ci
npm run dev
npm run typecheck
npm run build
```

Vite 固定使用 `5173` 端口。若浏览器不是当前 Vue 界面，确认命令在 `webUI/` 执行，且打开的是本次开发服务器。

在仓库根目录校验固定工程结构：

```bash
node scripts/validate-p1-regression-fixture.mjs
```

此脚本只校验 schema（工程结构版本）和内嵌资产格式，不替代浏览器端到端验证。

## 最小离线回归

1. 校验固定样例，启动 `webUI/` 开发服务器。
2. 在任意标签页顶部导入 `fixtures/p1-minimal-regression-project-v3.json`。
3. 确认资源库恢复 `测试提示音`、`测试BGM`、`旁白`、4 个默认滤波器和 8 个系统情绪。
4. 在音色、SFX 和 BGM 资源卡上试听，确认显示真实波形与播放进度；编辑 SFX/BGM，拖动两端裁剪手柄并保存，刷新后确认裁剪值和资产仍可恢复。
5. 在“脚本制作”确认脚本为 `P1 最小回归验收工程`，且有 `BGM play`、`bgImage`、两条 `dialogue`、`BGM stop` 五个块。
6. 预览两条内嵌台词，确认台词真实波形、裁剪手柄和背景图可恢复；刷新后再次确认资产从 IndexedDB（浏览器本地数据库）恢复。
7. 导出 `SRT`、`WAV`；浏览器支持 WebCodecs（网页编解码接口）且运行时可用时，再导出 `MP4`。

旧版选择器和历史基线保留在 [`P1-最小回归验收.md`](../P1-最小回归验收.md) 与 [`webUI/P0-手动回归入口.md`](../webUI/P0-手动回归入口.md)。Vue 工作台的当前回归应以本页步骤和下方最新记录为准。

# 本地开发与回归

## 正确入口

当前工作台入口是根目录 [`index.html`](../index.html)，使用 Vue 3 CDN 单文件实现；`webUI/` 已废弃。固定工程样例是 [`fixtures/p1-minimal-regression-project-v3.json`](../fixtures/p1-minimal-regression-project-v3.json)。

## 开发命令

```bash
python3 -m http.server 5173
```

打开 `http://127.0.0.1:5173/index.html`。修改单文件脚本后执行语法检查：

```bash
node -e "const fs=require('fs');const html=fs.readFileSync('index.html','utf8');for(const match of html.matchAll(/<script(?:\\s[^>]*)?>([\\s\\S]*?)<\\/script>/g)){if(match[1].trim())new Function(match[1]);}"
```

在仓库根目录校验固定工程结构：

```bash
node scripts/validate-p1-regression-fixture.mjs
```

此脚本只校验 schema（工程结构版本）和内嵌资产格式，不替代浏览器端到端验证。

## 最小离线回归

1. 校验固定样例，从仓库根目录启动静态文件服务器。
2. 在任意标签页顶部导入 `fixtures/p1-minimal-regression-project-v3.json`。
3. 确认资源库恢复 `测试提示音`、`测试BGM`、`旁白`、4 个默认滤波器和 8 个系统情绪。
4. 在音色、SFX 和 BGM 资源卡上试听，确认显示真实波形与播放进度；编辑 SFX/BGM，拖动两端裁剪手柄并保存，刷新后确认裁剪值和资产仍可恢复。
5. 在“脚本制作”确认脚本为 `P1 最小回归验收工程`，且有 `BGM play`、`bgImage`、两条 `dialogue`、`BGM stop` 五个块。
6. 预览两条内嵌台词，确认台词真实波形、裁剪手柄和背景图可恢复；刷新后再次确认资产从 IndexedDB（浏览器本地数据库）恢复。
7. 导出 `SRT`、`WAV`；浏览器支持 WebCodecs（网页编解码接口）且运行时可用时，再导出 `MP4`。

历史选择器和基线保留在 [`P1-最小回归验收.md`](../P1-最小回归验收.md) 与 [`webUI/P0-手动回归入口.md`](../webUI/P0-手动回归入口.md)。当前回归以根目录单文件入口为准。

## 最小在线回归

1. 按 [TTS-and-VoiceDesign 接入](TTS-and-VoiceDesign接入.md) 配置可访问的 TTS 服务。
2. 在 Prompt 管理中检查角色音色分析和动态参考文本设置；已有旧版自定义固定模板的浏览器应继续保持固定模式。
3. 分析角色音色并通过 Qwen 或 MiMo 生成参考音频，确认音色库中的 `promptText` 与实际提交文本一致。
4. 选择已绑定参考音色的台词；参考文本克隆需填写准确的参考音频文本。
5. 清除一条台词音频后重新生成，确认可播放；刷新后确认仍可恢复。
6. 按改动范围补充验证批量生成、顺序播放、停止播放和导出链路。

在线回归依赖实际运行的 `TTS-and-VoiceDesign` 服务、模型权重和本机算力，因此不包含在上述离线自动化记录中。
